\emph{Goal:} Learn the Backflow Algorithm and implement the Critical  Path Algorithm. \\

\begin{enumerate}
\item \fbox{Backflow Algorithm} 
	\begin{enumerate}
	\item Introduce an ``end" vertex, say $X$,  with a time of $[0].$
	\item From $X,$ move back through every vertex assigning it the maximum time to reach vertex $X$.
	\end{enumerate}
